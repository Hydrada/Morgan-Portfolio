# Public Portfolio Safety Rules

This repository is for public portfolio presentation only.

Do not commit:

- HydrantLoop app source code
- Testing app URLs or tester access details
- NDA material
- Credentials, API keys, secrets, or `.env` files
- Internal SOPs, unpublished calculations, or implementation recipes
- Private customer, tester, utility, or field records
- Local machine paths or private file exports

Safe content:

- Public project summaries
- Public HydrantLoop marketing links
- High-level architecture descriptions
- Sanitized screenshots
- Case studies focused on problem, approach, and outcome
